# 📊 Data Analyst Portfolio Projects

This repository contains a collection of data analysis projects focused on real-world datasets, data cleaning, exploratory data analysis (EDA), and data visualization using Python, Pandas, Matplotlib, Seaborn, and SQL.

---

## 🚀 Projects Overview

### 🏥 Hospital Patients Analytics Dashboard
A healthcare analytics dashboard that provides insights into patient records, admission trends, disease distribution, and key performance metrics. The project includes interactive visualizations and data filtering features for better decision-making.

🔗 Live Demo: https://hospitalpatientsdasboard.netlify.app/  
🔗 GitHub: https://github.com/kawsar334/data-analytics-portfolio/tree/main/hospitalPatient  

---

### 🎓 Student Performance Analysis
An exploratory data analysis project focused on understanding student academic performance. The dataset was cleaned and analyzed to identify how different factors influence exam scores and learning outcomes.

🔗 GitHub: https://github.com/kawsar334/data-analytics-portfolio/tree/main/studentPerformanceFactors  

---

### 🛒 Customer Shopping Behavior Analysis
A data analysis project that explores customer shopping patterns. The dataset was cleaned, transformed, and stored in SQL. Insights were generated using Matplotlib and Seaborn to understand customer behavior and trends.

🔗 GitHub: https://github.com/kawsar334/data-analytics-portfolio/blob/main/customer_shoping_behaviorsss/customer_shoping_behaviors.ipynb
